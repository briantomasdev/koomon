Core Capabilities

Everything your team needs.
Nothing you don't.

End-to-End Encrypted

NaCl-boxed CRDT envelopes. The relay server is a dumb pipe — it never sees your plaintext, period.

Local-First SQLite

Every read and write hits local SQLite first. The network is a sync mechanism, never a requirement.

Unified Rust Core

One koomon-core crate runs identically on macOS, Windows, Linux, iOS, and Android — zero code drift.

Y-CRDT Offline Sync

Conflict-free replicated data types keep message history consistent across devices — even after days offline.

On-Device AI Search

SwarmState AI indexes your message history locally. Semantic search across years of channels — zero cloud calls.

Native Mobile Push

Silent APNs and FCM wakes the Rust core on demand. Background sync that works within iOS and Android lifecycle limits.
